## 1. 设计目标

### 1.1 核心目标

| 目标 | 说明 |
|------|------|
| **类似 Monday.com 体验** | 现代化的项目管理界面，表格视图为主 |
| **业务导向** | 专为会计事务所工作流程设计 |
| **高效操作** | 快速查看、筛选、更新工作状态 |
| **多业务类型支持** | ITR / BAS / Payroll / Bookkeeping 等不同业务有不同视图 |

### 1.2 目标用户

| 角色 | 主要操作 |
|------|---------|
| **Preparer** | 查看分配的工作、更新状态、记录备注 |
| **Reviewer** | 审核工作、添加审核意见 |
| **Partner** | 总览所有工作、分配任务、查看报表 |
| **Admin** | 管理客户、配置系统 |

│  │ [All] [ITR] [BAS] [Payroll] [Bookkeeping] [Grants] ...     │ │
